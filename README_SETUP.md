## 🔧 How to Test

### 1. Register as Farmer
1. Go to `/register`
2. Select "Farmer" role
3. Complete registration

### 2. Add Products
1. Login as farmer
2. Go to farmer dashboard
3. Click "Add Product"
4. Fill in all details
5. Upload product image
6. Submit

### 3. Browse as Buyer
1. Register as "Buyer"
2. Go to buyer dashboard
3. Browse products added by farmers
4. Filter and search
5. Add to cart
